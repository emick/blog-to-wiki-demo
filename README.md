# Blog to Wiki Demo

This project converts blog content to a wiki format using MkDocs.

## Setup

1. Clone this repository

## Local Development

To run the site locally:

```bash
uv run mkdocs serve
```

This will start a local server.

## Test run

```bash
# Download blog
uv run download-blog.py

# Convert to wiki
uv run convert-to-wiki.py
```
